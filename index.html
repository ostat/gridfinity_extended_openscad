About Gridfinity Extended

Gridfinity Extended is an OpenSCAD library originally based on Jamie's library, built with added base features and a customizable extended feature set. It follows the Gridfinity specifications while adding powerful customization options.

Key Features:

  • Base Improvements: Corner-only magnets, flat base options, customizable finger slides, and configurable lip styles
  • Wall Customization: Patterned walls (grid, hex, voronoi, kumiko), wall cutouts, and tapered corners
  • Advanced Options: Subdivisions with custom configurations, removable divider walls, and sliding lids
  • Extended Components: Item holders, dividers, trays, drawers, and custom cutouts
  • Optimization Features: Efficient floor patterns, light bin options, and split bin support for larger prints
